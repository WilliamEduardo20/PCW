O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilização utilizada para definir a apresentação de elementos HTML em uma página web.

Tipos de CSS

  • CSS Incorporado 
    • <style></style>
  • CSS Importado 
    • <link rel="stylesheet" href="#">
  • CSS Inline 
    • <p style="color: red;">Texto</p>

Valores

  • Valores Absolutos
    • cm (centimeters)
    • mm (millimeters)
    • in (inches)
    • pt (points)
  • Valores Relativos
    • em (unidades relativas ao tamanho da fonte)
    • ex (unidades relativas à altura x da fonte)
    • px (pixels)

Texto

  • Alinhamento
    • text-align
      • left, right, center, justify
  • Decoração
    • text-decoration
      • none
      • underline
      • overline
      • line-through
  • Transformação
    • text-transform
      • none
      • capitalize
      • uppercase
      • lowercase
  • Outros
    • letter-spacing (espaçamento entre letras)
    • line-height (altura da linha)
    • text-indent (recuo de parágrafo)
    • direction (direção do texto)
      • ltr (da esquerda para a direita)
      • rtl (da direita para a esquerda)
    • color (cor do texto)
  • Sombra
    • text-shadow (sombra do texto)
      • text-shadow: offset-x offset-y blur-radius color;
      • offset-x (sombra horizontal)
      • offset-y (sombra vertical)
      • blur-radius (distância da sombra)
      • color (cor da sombra)

Bordas

  • border-style
    • dotted
    • solid
    • double
    • groove
    • ridge
    • inset
    • outset
  • border-color: color
  • border-spacing: valor
  • border-radius: valor
  • box-shadow
    • (1) posição horizontal
    • (2) posição vertical
    • (3) distância da sombra
    • (4) cor da sombra
  • border-collapse
    • collapse (as bordas das células se fundem)
    • separate (as bordas das células permanecem separadas)
  • border: 1px solid black

Display

  • block - Faz o elemento ocupar toda a largura disponível, começando em uma nova linha.
  • inline - Faz o elemento ser exibido em linha com outros elementos inline.
  • inline-block - Combina as características dos elementos block e inline.
  • none - Oculta o elemento e ele não ocupa espaço na página.
